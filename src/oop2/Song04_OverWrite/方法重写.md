###方法的重写  overwrite

```
> 为什么需要方法的重写
    子类在继承父类以后，获取了父类中声明的所有方法，但是可能不太适用于子类
    
> 何为方法重写
    子类需要对父类继承过来的方法进行覆盖、覆写操作

> 方法重写应遵循的规则
    方法名和形参列表必须相同
    权限修饰符不小于父类被重写的方法的权限修饰符
        不能重新父类中声明为private权限修饰的方法
    返回值类型
        父类被重写的方法返回值为 void 类型，则子类返回值类型必须为 void
        父类被重写的方法返回值为基本数据类型，则重写的方法返回值类型必须相同
        父类被重写的方法返回值为引用数据类型，则重新的方法返回值类型 可以与父类相同或者为被重写方法的返回值类型的子类
    ```
